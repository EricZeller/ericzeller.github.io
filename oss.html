Software Tools

For software development, I work with various programs:

  • IDEs:
    • Visual Studio Code: A powerful, cross-platform code editor used for building this website.
    • Android Studio: An official IDE for the Android platform used for developing Android apps.
    • IntelliJ IDEA Community Edition: A Java Integrated Development Environment (IDE) used for Java application development.
    • PyCharm Community Edition: A Python IDE used for Python application development.
    • Flutter: A Google UI toolkit for building natively compiled applications for mobile, web, and desktop from a single codebase.
    • Nano: A simple command line text editor, used for quickly edit files, configs etc. on Linux.
    • Xcode: An IDE for macOS I am using for compiling software for macOS, iOS, iPadOS, watchOS, and tvOS.

  • Other tools:
    • Quickemu: A tool for quickly creating macOS virtual machines (VMs) on Linux hosts for software development and testing purposes.
    • Adobe Photoshop: A software for editing and creating graphics, including icons and other graphical elements.
